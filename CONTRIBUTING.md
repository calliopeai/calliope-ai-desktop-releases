# Contributing to Calliope AI Desktop Applications

Thank you for your interest in contributing to Calliope AI's desktop applications!

## Reporting Issues

This repository is specifically for **release distributions** of our desktop applications.

### Where to Report Issues

**For application bugs and feature requests:**

- **Calliope AI IDE issues**: Please report to the main IDE repository
- **Calliope AI Lab issues**: Please report to the main Lab repository

**For release/download issues:**

If you're experiencing problems with:
- Download links
- Installation instructions
- Documentation in this repository
- Release artifacts

Please [open an issue](../../issues) in this repository.

## What to Include in Bug Reports

When reporting issues, please include:

1. **Application & Version**: Which app and version you're using
2. **Operating System**: OS name and version
3. **Architecture**: x64, ARM64, etc.
4. **Package Format**: .zip, .deb, .exe, etc.
5. **Steps to Reproduce**: Detailed steps to recreate the issue
6. **Expected Behavior**: What you expected to happen
7. **Actual Behavior**: What actually happened
8. **Logs**: Any relevant error messages or log files

## Release Management

This repository is maintained by the Calliope Labs team for distributing official releases.

Release artifacts are automatically generated from our build systems and published here.

## Documentation Improvements

We welcome improvements to:
- README documentation
- Installation instructions
- Troubleshooting guides
- Platform-specific guidance

To suggest documentation improvements:
1. Fork this repository
2. Make your changes
3. Submit a pull request with a clear description

## Code of Conduct

Please be respectful and constructive in all interactions. We're all here to build great tools together.
